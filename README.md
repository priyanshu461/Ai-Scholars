# Ai Scholars ERP (Next.js)

A lightweight but full-featured ERP starter built with **Next.js 15**, **React 19**, and **TypeScript**.

## Included ERP foundations

- Dashboard with KPI cards
- Modular ERP areas (CRM, Finance, Inventory, HR, Projects, Analytics)
- Sidebar navigation for expansion
- Modern responsive dark UI

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
